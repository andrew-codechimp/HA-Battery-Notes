    Blueprints#

    Blueprints are an excellent way to get you up and running with the integration quickly. They can also be used as a guide for setting up new automations which you can tailor to your needs.

    Battery Threshold#

    Install blueprint | Source

    This blueprint will allow notifications to be raised and/or custom actions to be performed when the battery threshold is met.
    It is extended from the community Battery Low Notification automation yaml for those who'd prefer an easy way to get started.

    Battery Replaced#

    Install blueprint | Source

    This blueprint will automatically update the battery replaced sensor and custom actions to be performed when the battery increases.
    It is extended from the community Battery Replaced automation yaml for those who'd prefer an easy way to get started.

    Battery Not Reported#

    Install blueprint | Source

    This blueprint will allow notifications to be raised and/or custom actions to be performed when the battery not reported event is fired.
    It is extended from the community Battery Not Reported automation yaml for those who'd prefer an easy way to get started.

    Info

    You must trigger the check_battery_not_reported action via an automation to raise events, see Check Battery Last Reported Daily.
